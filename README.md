## 在普通对战图上的修改
- 大部分玩家单位和英雄都缩小了50%，移动速度和攻击距离也都做了相应的调整
- 每个种族的部队都做了简化，都只有一个兵营，可以建造四种兵。
- 每个种族都额外添加了两个英雄，一共6个英雄
- 每个英雄都有五个技能，3个可以升4级的小技能，1一个可以升两级的大招和一个只有一级的终极技能。
- 支持ai对战，ai会随机选择所有英雄，随机选择英雄技能的升级路线。每个技能都会使用。
- ai会把多余的金币和木头全部建造成防御塔或月亮井。
